<div style={{ height: '32px'}}/>
<p className='text'>The 4px base unit forms the basis of our space token system, as the base unit spacing.100. Every space token is a multiple of this base unit, the number suffix representing the percentage of the base unit.</p>
<p className='text'>For example, spacing-200 is 200% of the size of the base unit, therefore represents 16px.</p>
<p className='text'>Each space token should be used in place of the raw pixel or REM values when adding space between components or objects on a page. </p>
